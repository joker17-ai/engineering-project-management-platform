# 蓝箭项目管理平台

基于“工程项目管理1.1”的工程结构思路，在 `工程项目管理1.2` 中重新建立的公众项目管理平台。

## 数据仓库

- GitHub: https://github.com/joker17-ai/engineering-project-management-platform
- Clone: https://github.com/joker17-ai/engineering-project-management-platform.git
- 建议数据目录：
  - `data/projects`
  - `data/documents`
  - `data/progress`
  - `data/public-notices`

## 本地预览

```powershell
npm install
npm start
```

也可以直接用任意静态服务器打开本目录。
